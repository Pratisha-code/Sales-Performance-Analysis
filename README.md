# Brand Sales Analysis Dashboard (2024–2025)

## Project Overview

This project analyzes sales performance across multiple technology brands and product categories.

The objective is to identify top-performing brands, high-demand products, regional sales patterns, and profitability trends using Excel and Power BI.

## Business Problem

Organizations often struggle to understand which brands and product categories contribute most to revenue. 

This project helps stakeholders make data-driven decisions regarding inventory management, sales strategy, and market expansion.

## Project Objective

To analyze brand-wise sales performance and uncover actionable insights that improve revenue growth, inventory planning, and business decision-making.

## Dataset Description

The dataset contains over 5,000 sales transactions with the following attributes:

- Order Number

- State Code
  
- Customer Name
  
- Order Date
 
- Status
  
- Product Category

- Brand

- Cost

- Sales

- Quantity

- Total Cost

- Total Sales

- Assigned Supervisor
